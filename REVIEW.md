# [Feature] 商品レビュー機能追加（評価集計機能含む）

| 項目 | 内容 |
|---|---|
| 概要 (Overview) | 購入者が商品の評価・レビューを投稿できる機能を追加する。注文詳細画面からレビュー入力画面へ遷移し、商品詳細画面・トップ画面で他ユーザーのレビュー情報を確認できるようにする。 |
| 目的 | 購入者の評価を商品情報へ反映し、購入判断材料を増やす。また、不正レビューや品質確認の参考情報として利用できるようにする。 |
| 詳細仕様 | 注文詳細画面の商品名横に「レビューを書く」ボタンを追加する。レビュー入力画面では評価（★1〜5）とコメントを登録できる。コメントは未入力でも登録可能とする。 |
| 投稿条件 | レビュー投稿可能なのは、対象商品を購入済みかつ注文ステータスが発送完了済みのユーザーのみとする。 |
| レビュー仕様 | 評価：1〜5段階（必須）、コメント：任意（null可）、投稿日時：システム日時で自動登録する。 |
| レビュー登録仕様 | 同一ユーザーが同一商品へ複数回レビュー投稿可能とする。購入ごとにレビュー投稿可能。 |
| 編集仕様 | 登録後のレビュー編集機能は実装しない。投稿内容変更は不可とする。 |
| 削除仕様 | レビュー削除は投稿した本人、および管理者（ADMIN）のみ可能とする。 |
| 商品評価集計 | 商品ごとのレビュー平均評価を計算し、平均★数とレビュー件数を表示する。 |
| 商品詳細画面表示 | 商品詳細画面下部に平均評価、レビュー件数、レビュー一覧を表示する。 |
| トップ画面表示 | 商品一覧の商品情報に平均評価（★表示）とレビュー件数を表示する。 |
| 評価表示例 | ★★★★☆ 4.2 / レビュー件数：35件 のように表示する。 |
| レビュー並び替え | レビュー一覧は「新しい順」を初期値とする。「評価が高い順」「評価が低い順」へ変更可能とする。 |
| 表示内容 | レビューには投稿者名、評価（★表示）、コメント、投稿日を表示する。 |
| 表示件数 | 商品詳細画面では最新5件を表示する。「全件表示」ボタン押下で全レビューを表示する。 |
| 技術的制約・前提条件 (Constraints & Context) | Spring Boot MVC構成を維持する。Spring Data JPAを利用する。画面はThymeleafを利用する。ControllerからRepositoryを呼び出す既存構成に合わせる。 |
| 制約 | Service層は新規追加しない。既存Entity・Controller・Repository構成を拡張する。 |
| DB変更 | 新規Reviewテーブルを追加する。 |
| Reviewテーブル項目 | id、user_id、product_id、order_id、rating、comment、created_dateを保持する。 |
| Entity追加 | Review Entityを追加し、User・Product・Orderと関連付ける。 |
| Repository追加 | ReviewRepositoryを追加し、商品ID指定取得、平均評価取得、件数取得、並び替え取得処理を実装する。 |
| 受け入れ条件 (Acceptance Criteria) | 注文詳細画面の商品名横に「レビューを書く」ボタンが表示されること。 |
| 受け入れ条件 | 発送完了済みの商品だけレビュー投稿できること。 |
| 受け入れ条件 | 未購入ユーザーはレビュー投稿できないこと。 |
| 受け入れ条件 | 評価は1〜5のみ登録可能であること。 |
| 受け入れ条件 | コメント未入力でもレビュー登録できること。 |
| 受け入れ条件 | 商品ごとの平均評価が正しく計算されること。 |
| 受け入れ条件 | 商品詳細画面に平均評価とレビュー件数が表示されること。 |
| 受け入れ条件 | トップ画面の商品一覧に平均評価が表示されること。 |
| 受け入れ条件 | 最新5件のレビューが商品詳細画面に表示されること。 |
| 受け入れ条件 | 全件表示ボタンから全レビュー確認できること。 |
| 受け入れ条件 | 新しい順・評価高い順・評価低い順で並び替えできること。 |
| 受け入れ条件 | 投稿者本人のみ自分のレビューを削除できること。 |
| 受け入れ条件 | ADMIN権限ユーザーは全レビュー削除可能であること。 |
| 受け入れ条件 | レビュー編集機能が存在しないこと。 |

| 実装ステップ案 | ⑤ レビュー入力画面作成 |
| 実装ステップ案 | ⑥ レビュー登録処理追加 |
| 実装ステップ案 | ⑦ 商品詳細画面へレビュー一覧追加 |
| 実装ステップ案 | ⑧ 商品単位の平均評価取得処理追加 |
| 実装ステップ案 | ⑨ トップ画面の商品一覧へ評価表示追加 |
| 実装ステップ案 | ⑩ 削除権限チェック追加 |
| テスト確認項目 | レビュー投稿、購入制限、評価登録、平均評価計算、商品詳細表示、トップ表示、並び替え、本人削除、ADMIN削除確認。 |
